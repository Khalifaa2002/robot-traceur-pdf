# 🤖 Robot Traceur de Plan PDF

Un robot autonome qui trace des plans PDF sur terrain avec correction en temps réel.

## 🚀 Caractéristiques
- 📖 Extraction PDF
- 🎯 Génération trajectoire
- 📍 Localisation odométrie
- 🔄 Suivi trajectoire PID
